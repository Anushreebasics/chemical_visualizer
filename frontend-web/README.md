# Chemical Equipment Visualizer - Frontend (Web)

React.js based web application for chemical equipment data visualization.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## Features

- User registration and authentication
- CSV file upload
- Real-time data visualization with Chart.js
- Equipment type distribution charts
- Upload history tracking
- PDF report generation

## Project Structure

- `src/pages/` - Application pages (Login, Register, Dashboard)
- `src/components/` - Reusable components (CSVUpload, Charts, History, etc.)
- `src/api.js` - API integration with Django backend
